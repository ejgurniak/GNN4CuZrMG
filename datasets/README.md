### Explanation of the dataset
- structure files are in POSCAR format
- the first line is the class label
- the rest of the file describes the structure of the sample
#### class labels
- "0": ~10<sup>15</sup> K/s
- "1": 10<sup>13</sup> K/s
- "2": 10<sup>12</sup> K/s
- "3": 10<sup>11</sup> K/s
- "4": 10<sup>10</sup> K/s
- "5": 10<sup>9</sup> K/s
